# README更新コマンド

リポジトリの現在の状態を確認し、ルートのREADME.mdを最新の状態に更新してください。

## 実行手順

1. リポジトリ内のすべてのディレクトリとファイルを確認
2. 各サブディレクトリのREADME.mdを読み込み、内容を把握
3. 新規追加・削除されたリソースを検出
4. ルートのREADME.mdを以下の構成で更新:
   - ディレクトリ構成（ツリー形式）
   - 各リソースの概要と詳細READMEへのリンク
   - クイックスタート（基本的な使用コマンド）
   - ファイル間の依存関係
   - コーディングエージェント向け情報

## 更新時の注意点

- 既存の説明を可能な限り維持する
- 新規リソースは適切なカテゴリに追加
- 削除されたリソースはREADMEからも削除
- 日本語で記載する
